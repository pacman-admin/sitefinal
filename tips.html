Windows

Note: I have no knowledge of Windows 11. I only know Windows 10 and earlier.

Shift+Delete -> Delete file (insecure)

F5 -> Reload

Ctrl+Shift+Esc -> Open Task Manager

Cmd+Option+V -> Cut in Finder (See Description below)

In Finder, you can copy a file with Cmd+C and Paste with Cmd+V. To cut a file and paste it, Press Cmd+C then Cmd+Option+V to paste and delete from origin (cut). Try out this app: Command X by Sindre Sorhus (Requires MacOS 13 or later)
