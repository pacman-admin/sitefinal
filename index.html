Info and Installation help

To install Pac-Man for Microsoft Windows, download the desired version, Double click the .exe file, and follow the installer's instructions.

To install pac-man on macOS, copy the .app file to /Applications

Note: While Microsoft and Apple are suspicious of Pac-Man, it is not a virus. You may scan it with an antivirus program to confirm this.
