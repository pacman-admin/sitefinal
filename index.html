To install Pac-Man for Microsoft Windows, download an OpenJDK binary and then download the desired version. Double click the .exe file and follow the installer's instructions.

To install on Mac, download Java 8 and then the desired version. Double-click the downloaded .zip file, open the created folder, and double click the Pac-Man file.
