How to Update Pac-Man on Windows when Installed in Program Files (Administrative install)

A step-by-step guide to updating pac-man.

Prerequisites

Install scoop

Go to scoop.sh

Follow instrustions on the website to install scoop.

Install sudo

Press Windows Key + R

Type cmd

In the window that opens, type "scoop install sudo"

All done!

Procedure

Download the new version

In the update window, click "Download New Version"
