MacOS Versions

MacOS has a new Gatekeeper system that effectively renders the program useless, I have not been able to find the reason behind this. To run pac-man, you must disable Gatekeeper. PROCEED AT YOUR OWN RISK: to disable Gatekeeper, open Terminal and type `sudo spctl --master-disable`, you will need to have admin privlages to proceed. DISABLING GATEKKEEPER LEAVES YOUR MAC MORE VULNERABLE TO POTENTIAL VIRUSES. PROCEED WITH CAUTION.

To run Pac-Man, without disabling Gatekeeper, you must open "application.macosx", control-click "Pac_Man.app", click "Show Package Contents", open the "Contents" folder, and finally open the "MacOS" folder and double-click the file inside it. Please email me at help@getpacman.gq if you encounter any problems.
